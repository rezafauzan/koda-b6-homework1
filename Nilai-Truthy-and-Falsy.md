# Truthy And Falsy

## Definisi
Truthy and Falsy adalah suatu nilai yang dianggap True/Benar(Truthy) dan nilai yang dianggap False/Salah(Falsy)

## Nilai Nilai Truthy
yang termasuk nilai Truthy biasanya adalah sebagai berikut ini :
|function my(){}                 | tipe data non-primitf akan menghasilkan true meskipun kosong
|const array = []                | tipe data non-primitf akan menghasilkan true meskipun kosong
|const object = {}               | tipe data non-primitf akan menghasilkan true meskipun kosong
|true                            |
|let strs = 'String Tidak Kosong'|
|let num = 1                     |

## Nilai Nilai Falsy
yang termasuk nilai Falsy biasanya adalah sebagai berikut ini :
|false        |
|let strs = ''|
|let num = 0  |
|null         |
|undefined    |
|NaN          |
